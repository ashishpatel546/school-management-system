# School Management System

A comprehensive web application for managing school data, including teachers, students, classes, and enrollments.

## Features
- **Teachers**: Manage profiles, assign subjects, and designate Class Teachers.
- **Students**: Enrollments, subject selection, and class promotion.
- **Classes**: Manage sections and class teachers.
- **History**: Track historical data for student movements and teacher assignments.

## Tech Stack
- **Frontend**: Next.js 13+, TypeScript, TailwindCSS.
- **Backend**: NestJS, TypeORM, PostgreSQL (or similar).

## Getting Started

1.  **Clone the repository**:
    ```bash
    git clone git@github.com:ashishpatel546/school-management-system.git
    ```

2.  **Install dependencies**:
    ```bash
    # Backend
    cd backend
    npm install

    # Frontend
    cd frontend
    npm install
    ```

3.  **Run the application**:
    ```bash
    # Custom script or separate terminals
    # Terminal 1
    cd backend && npm run start:dev

    # Terminal 2
    cd frontend && npm run dev
    ```
